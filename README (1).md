## Descrição do Projeto
Este projeto consiste em um jogo de Batalha Naval desenvolvido em linguagem C, executado em modo texto (terminal).  
O objetivo do jogo é localizar e destruir todos os navios posicionados aleatoriamente no tabuleiro.

## Funcionalidades
- Tabuleiro 5x5
- Posicionamento aleatório de 3 navios
- Entrada de dados pelo usuário (linha e coluna)
- Indicação de acerto ou erro
- Finalização do jogo após destruir todos os navios

## Como compilar e executar
```bash
gcc main.c -o batalha_naval
./batalha_naval
